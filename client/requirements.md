## Packages
framer-motion | For smooth retro UI animations and modal transitions
howler | For retro sound effects
usehooks-ts | For keyboard event listeners and interval handling

## Notes
- Game uses a CSS-grid or relative positioning system for the 2D map.
- Assets will be generated using CSS shapes or simple placeholder pixel art if no images are provided.
- Keyboard controls (WASD/Arrows) are handled via global event listeners.
- API endpoints /api/content and /api/progress are assumed to exist based on schema.
